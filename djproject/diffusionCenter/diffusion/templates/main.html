Diffusion Center

Step 1: Define a Profile

A profile consists of the order and thicknesses of the materials (generally metals)that are deposited onto one another. The program currently simulates profiles as infinite-area finite-depth sheets sequentially placed onto one another.

In order to define a profile, click setup profile.

Step 2: Define an Annealing Process

An annealing process consists of a series of temperatures the material profile is subjected to. The allowed temperature ranges are limited by those of the materials involved.

In order to define an annealing process, click setup process.

Step 3: Run a Profile Thorugh a Process

A profile can be run through a process as long as the temperature ranges are valid for each of its materials, and all diffusion relations between materials in the profile are defined (or fallback values are provided).

Results can be saved, and graphs are produced to show the diffusion.

In order to run a profile through an annealing process, click run process.

Step 4: View Anneal Results

Results of anneal-induced diffusion can be saved, and graphs are produced to show the diffusion.

In order to view saved anneal results, click saved results.
